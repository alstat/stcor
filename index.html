    Description

    The stcor package provides functions for correlograms of the advanced time series model, the Space-Time Autoregressive Integrated Moving Average (STARIMA). These functions are the Space-Time Autocorrelation Function (stacf) and Space-Time Partial Autocorrelation Function (stpacf). Including function for centralizing (stcenter) the data to its mean.

    Installation

    if (!require("devtools"))
  install.packages("devtools")
devtools::install_github("stcor", "alstat")

    Vignette

    To start with, let us simulate a space-time series. Assuming there are seven (7) spatial locations with 155 observation. Then we have,

    series <- arima.sim(list(order = c(2,0,0),
                    ar = c(0.7, -0.5)), n = 155)

location1 <- series + rnorm(length(series))
location2 <- series + rnorm(length(series))
location3 <- series + rnorm(length(series))
location4 <- series + rnorm(length(series))
location5 <- series + rnorm(length(series))
location6 <- series + rnorm(length(series))
location7 <- series + rnorm(length(series))

    Now we will merge this into a data frame object,

    series.df <- data.frame(location1, location2, location3,
                        location4, location5, location6,
                        location7)

    And it is required to centralize the data to its mean,

    series.df <- stcenter(series.df)

    Then assuming the spatial neighbourhood of our data has the following weights matrices,

    • Weights matrix for first spatial order
    W1 <- rbind(c(0,0,0,0,0,1,0),
            c(0,0,0,1,0,0,0),
            c(0,0,0,1,0,0,0),
            c(0,1/3,1/3,0,0,1/3,0),
            c(0,0,0,0,0,1/2,1/2),
            c(1/4,0,0,1/4,1/4,0,1/4),
            c(0,0,0,0,1/2,1/2,0))
    • Weights matrix for second spatial order
    W2 <- rbind(c(0,0,1/3,1/3,0,0,1/3),
            c(0,0,1/2,0,0,1/2,0),
            c(1/3,1/3,0,0,0,1/3,0),
            c(1/3,0,0,0,1/3,0,1/3),
            c(0,0,0,1,0,0,0),
            c(0,1/2,1/2,0,0,0,0),
            c(1/2,0,0,1/2,0,0,0))
    • Weights matrix for third spatial order
    W3 <- rbind(c(0,1/2,0,0,1/2,0,0),
            c(1/3,0,0,0,1/3,0,1/3),
            c(0,0,0,0,1/2,0,1/2),
            c(0,0,0,0,0,0,0),
            c(1/3,1/3,1/3,0,0,0,0),
            c(0,0,0,0,0,0,0),
            c(0,1/2,1/2,0,0,0,0))
